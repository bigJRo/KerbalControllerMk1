# ILI9341_t3 fonts (converted from sumotoy tFont)

These `.c` files are the KCMk1 display fonts converted from the original sumotoy
`tFont` format (in `../fonts/`) to **ILI9341_t3** format, for the wwatson4506
RA8876 GFX text engine used on the 7" TFT (hardware rev 2).

- Generated by `tfont_to_ili9341.py` (vendored here for reproducibility):
  ```
  python3 tfont_to_ili9341.py ../fonts ./
  ```
- The converter **round-trip verifies** every glyph: it decodes its own output
  with the ILI9341_t3 `drawFontChar` bit logic and asserts the bitmap matches the
  source pixel-for-pixel. Conversion aborts on any mismatch.
- Glyphs are emitted full-height and top-aligned (`height = font_height`,
  `yoffset = 0`, `cap_height = font_height`) so `origin_y = cursor_y` — preserving
  the old top-left cursor semantics of the sumotoy/RA8875 text helpers.
- `delta` (advance) = glyph width, matching the old `getFontStringWidth()` sum so
  text alignment math stays consistent.
- Special glyphs (Δ, °, µ, ¤, arrows/symbols 0x80–0x8d) are carried in a second
  index range (0x80–0xb5) with blank entries for unused codes.

`kcm_ili9341_font.h` supplies the `ILI9341_t3_font_t` type — it prefers the
installed library's definition (via `__has_include`) and falls back to the
canonical layout. See `../../../PORTING_7inch_TFT.md`.

Fonts: Roboto_Black_{12,16,20,24,28,32,36,40,48,72}, TerminalFont_{16,32}.
